# 작업순서

## 1단계: 테스트 지역 3개로 시작

`data/areas.csv`에 테스트 지역 3개만 넣고 시작합니다.

## 2단계: 서비스 2개 확인

`data/services.json`에서 하수구막힘과 누수탐지 설정을 분리합니다.

## 3단계: 템플릿 작업

`templates/hasugu.html`
- 하수구막힘 전용 문구
- 하수구 팜플렛
- 하수구 관련 FAQ

`templates/nusu.html`
- 누수탐지 전용 문구
- 누수 팜플렛
- 누수 관련 FAQ

## 4단계: HTML 자동 생성

```bash
npm run build
```

## 5단계: dist 확인

생성된 파일:

```txt
dist/hasugu/busan-bukgu-mandeokdong/index.html
dist/nusu/busan-bukgu-mandeokdong/index.html
```

## 6단계: Netlify 배포

GitHub에 올리고 Netlify에서 연결합니다.
